# cookies

Exemplo didático de tema claro/escuro usando cookies para persistir a preferência do usuário.

- `index.html`: interface refinada para escolha de tema.
- `styles.css`: design mais profissional e controles responsivos.
- `script.js`: lógica de leitura e gravação de cookie `themeMode` por 30 dias.

Abra `index.html` em um navegador para verificar o funcionamento. O tema é reaplicado automaticamente no carregamento quando o cookie estiver presente.
